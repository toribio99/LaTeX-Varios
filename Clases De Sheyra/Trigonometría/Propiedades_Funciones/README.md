# Guía de Propiedades de las Funciones

## Descripción

Guía educativa completa sobre las propiedades y características de las funciones para estudiantes de grado 10. Esta guía profundiza en el análisis de funciones, sus propiedades algebraicas y gráficas.

## Especificaciones

- **Título:** Propiedades de las Funciones
- **Autor:** Prof. Toribio De J Arrieta F
- **Institución:** La Pruebita
- **Grado:** 10
- **Asignatura:** Trigonometría
- **Páginas:** 41
- **Compilador:** LuaLaTeX
- **Fecha:** Noviembre 2025

## Contenido

### 1. Introducción (2 páginas)
- Importancia de las propiedades de funciones
- Análisis de comportamiento de funciones
- Aplicaciones en matemáticas y ciencias

### 2. Conceptos Fundamentales (25 páginas)

**Propiedades Algebraicas:**
- Función inyectiva (uno a uno)
- Función sobreyectiva (sobre)
- Función biyectiva
- Función inversa

**Propiedades de Simetría:**
- Funciones pares
- Funciones impares
- Simetría respecto al eje y
- Simetría respecto al origen

**Comportamiento de Funciones:**
- Funciones crecientes
- Funciones decrecientes
- Funciones constantes
- Intervalos de monotonía

**Funciones Acotadas:**
- Cota superior
- Cota inferior
- Funciones acotadas superiormente
- Funciones acotadas inferiormente

**Extremos de Funciones:**
- Máximos absolutos
- Mínimos absolutos
- Máximos relativos o locales
- Mínimos relativos o locales

**Funciones Periódicas:**
- Concepto de periodicidad
- Período fundamental
- Ejemplos de funciones periódicas

### 3. Ejemplos Resueltos (8 páginas)

1. **Análisis de simetría** - Determinar si funciones son pares o impares
2. **Intervalos de crecimiento** - Identificar dónde crece o decrece una función
3. **Funciones acotadas** - Determinar cotas superiores e inferiores
4. **Extremos** - Encontrar máximos y mínimos
5. **Funciones periódicas** - Calcular períodos fundamentales

### 4. Ejercicios Propuestos (2 páginas)
10 ejercicios variados sobre todas las propiedades

### 5. Soluciones Detalladas (10 páginas)
Soluciones completas con gráficas y análisis detallado

### 6. Ejercicios Inversos (1 página)
4 ejercicios desafiantes:
- Construcción de funciones con propiedades específicas
- Análisis de funciones compuestas
- Problemas de optimización

### 7. Soluciones de Ejercicios Inversos (6 páginas)
Soluciones detalladas con justificaciones matemáticas

### 8. Conclusión (1 página)
- Resumen de propiedades fundamentales
- Tabla de propiedades
- Aplicaciones prácticas

## Características

- **Análisis riguroso** de propiedades matemáticas
- **Gráficas detalladas** usando TikZ y pgfplots
- **Ejemplos visuales** para cada propiedad
- **Cajas de colores** para definiciones y teoremas
- **Demostraciones** de propiedades importantes
- **Aplicaciones** en problemas reales

## Paquetes LaTeX Utilizados

- `fontspec` - Manejo de fuentes
- `babel[spanish]` - Soporte para español
- `amsmath, amssymb` - Símbolos matemáticos
- `geometry` - Márgenes
- `xcolor` - Colores
- `tikz, pgfplots` - Gráficos y funciones
- `tcolorbox` - Cajas de colores
- `multicol` - Columnas múltiples
- `enumitem` - Listas personalizadas
- `titlesec` - Formato de secciones

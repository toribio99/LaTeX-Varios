# Guía de Ángulos: Fundamentos de Trigonometría

## Descripción

Guía educativa completa sobre ángulos para estudiantes de grado 10, diseñada como base fundamental para el estudio de la trigonometría. Esta guía aborda sistemas de medición, clasificación y operaciones con ángulos.

## Contenido

### 1. Introducción (2 páginas)
- Concepto de ángulo
- Importancia en trigonometría
- Elementos de un ángulo
- Aplicaciones en la vida real

### 2. Conceptos Fundamentales (20 páginas)

**Definiciones Básicas:**
- Ángulo como rotación
- Vértice y lados
- Posición estándar
- Lado inicial y lado terminal

**Sistemas de Medición Angular:**

**Sistema Sexagesimal:**
- Grados (°)
- Minutos (')
- Segundos ('')
- Conversiones entre unidades
- Operaciones (suma, resta)

**Sistema Circular (Radianes):**
- Definición de radián
- Relación entre grados y radianes
- Fórmulas de conversión
- Longitud de arco

**Sistema Centesimal:**
- Grados centesimales
- Conversiones

**Clasificación de Ángulos:**
- Ángulo agudo (0° < α < 90°)
- Ángulo recto (α = 90°)
- Ángulo obtuso (90° < α < 180°)
- Ángulo llano (α = 180°)
- Ángulo completo (α = 360°)

**Posición en el Plano Cartesiano:**
- Ángulos en posición estándar
- Cuadrantes (I, II, III, IV)
- Ángulos cuadrantales
- Ángulos positivos y negativos

**Relaciones entre Ángulos:**
- Ángulos complementarios (suman 90°)
- Ángulos suplementarios (suman 180°)
- Ángulos coterminales
- Ángulo de referencia

### 3. Ejemplos Resueltos (8 páginas)

1. **Conversión de unidades sexagesimales** - De grados a minutos y segundos
2. **Conversión grados-radianes** - En ambas direcciones
3. **Operaciones con ángulos** - Suma y resta en sistema sexagesimal
4. **Ángulos coterminales** - Encontrar ángulos coterminales
5. **Ángulos de referencia** - Cálculo en cada cuadrante
6. **Aplicación: Longitud de arco** - Problemas prácticos

### 4. Ejercicios Propuestos (2 páginas)

8 ejercicios cubriendo:
- Conversiones entre sistemas
- Operaciones con ángulos
- Clasificación
- Ángulos complementarios y suplementarios
- Ángulos coterminales
- Ángulos de referencia
- Aplicaciones prácticas

### 5. Soluciones Detalladas (12 páginas)
Soluciones paso a paso con diagramas y verificaciones

### 6. Ejercicios Inversos (1 página)

4 ejercicios desafiantes:
- Encontrar ángulos con propiedades específicas
- Problemas de aplicación en navegación
- Cálculos con arcos circulares
- Problemas de optimización angular

### 7. Soluciones de Ejercicios Inversos (6 páginas)
Soluciones completas con múltiples enfoques

### 8. Conclusión (1 página)
- Resumen de sistemas de medición
- Tabla de conversiones
- Clasificación de ángulos
- Preparación para funciones trigonométricas

## Notas Pedagógicas

Esta guía está diseñada para:

1. Proporcionar una base sólida sobre ángulos antes de estudiar trigonometría
2. Dominar las conversiones entre sistemas de medición angular
3. Comprender la posición de ángulos en el plano cartesiano
4. Desarrollar habilidades para trabajar con radianes
5. Preparar conceptos de ángulo de referencia y coterminales
6. Aplicar conocimientos en problemas prácticos de geometría
